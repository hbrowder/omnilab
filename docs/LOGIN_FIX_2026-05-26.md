## Lessons Learned

1. **Always verify backend is running before debugging auth issues**
   - Check process: `ps aux | grep python`
   - Check port: `ss -tuln | grep :5000`

2. **FastAPI trailing slash behavior**
   - Routes defined as `@router.get("/")` register as `/api/prefix/`
   - Frontend must match exactly: `/api/labs/` not `/api/labs`

3. **Port configuration lives in frontend code**
   - File: `frontend/src/utils/api.js`
   - Must rebuild frontend after changes
   - No environment variable override currently

4. **Password reset pattern for bcrypt**
   ```python
   from core.auth import hash_password, verify_password
   new_hash = hash_password("new_password")
   # Update DB: users.password_hash = new_hash
   verify_password("new_password", new_hash)  # Verify it works
   ```

3. **Password Reset Script**
   Create `scripts/reset_password.py` for easier admin recovery:
   ```python
   #!/usr/bin/env python3
   import sys
   from pathlib import Path
   sys.path.insert(0, str(Path(__file__).parent.parent / "backend"))
   
   from core.auth import hash_password
   import sqlite3, os
   
   username = input("Username: ")
   new_password = input("New password: ")
   
   db_path = os.path.expanduser('~/.omnilab/omnilab.db')
   conn = sqlite3.connect(db_path)
   password_hash = hash_password(new_password)
   conn.execute("UPDATE users SET password_hash = ? WHERE username = ?", 
                (password_hash, username))
   conn.commit()
   print(f"✅ Password updated for {username}")
   ```
